- **Go2**: `quat="-0.000543471 0.713435 -0.00173769 0.700719"`

## 4. 关节配置

### 4.1 关节轴定义

两种机器人使用相同的关节轴定义：
- **Hip 关节**: 轴 = (1, 0, 0)，绕 X 轴旋转（侧摆）
- **Thigh 关节**: 轴 = (0, 1, 0)，绕 Y 轴旋转（抬腿）
- **Calf 关节**: 轴 = (0, 1, 0)，绕 Y 轴旋转（膝关节）

### 4.2 关节角度限制

| 关节 | Aliengo | Go2 | 差异 |
|------|---------|-----|------|
| Hip 下限 | **-1.22173 rad** (-70°) | **-1.0472 rad** (-60°) | Go2 限制更严格 |
| Hip 上限 | **+1.22173 rad** (+70°) | **+1.0472 rad** (+60°) | Go2 限制更严格 |
| Front Thigh 下限 | **无限制** | **-1.5708 rad** (-90°) | Go2 有限制 |
| Front Thigh 上限 | **无限制** | **+3.4907 rad** (+200°) | Go2 有限制 |
| Rear Thigh 下限 | **无限制** | **-0.5236 rad** (-30°) | Go2 有限制 |
| Rear Thigh 上限 | **无限制** | **+4.5379 rad** (+260°) | Go2 有限制 |
| Calf 下限 | **-2.77507 rad** (-159°) | **-2.7227 rad** (-156°) | 相近 |
| Calf 上限 | **-0.64577 rad** (-37°) | **-0.83776 rad** (-48°) | Go2 可弯曲更多 |

**关键差异**: 
- Go2 的髋关节限制更严格（±60° vs ±70°）
- Go2 的前后腿大腿关节有不同的限制范围，前腿和后腿配置不对称
- Aliengo 的大腿关节无限制，Go2 有明确的限制范围

### 4.3 关节阻尼

| 机器人 | 阻尼系数 |
|--------|----------|
| Aliengo | 0.2 (URDF) / 0.01 (MJCF 默认 + 自定义) |
| Go2 | 0.2 (URDF) |

### 4.4 关节速度限制

| 机器人 | 限制 |
|--------|------|
| Aliengo | 40 rad/s (所有关节) |
| Go2 | 40 rad/s (所有关节) |

## 10. MJCF vs URDF 差异

### 10.1 Aliengo URDF vs MJCF 惯性一致性

| 连杆 | URDF 质量 | MJCF 质量 | 状态 |
|------|-----------|-----------|------|
| Trunk | 9.041 kg | 9.042 kg | 一致 ✓ |
| Hip | 1.993 kg | 1.993 kg | 一致 ✓ |
| Thigh | 0.639 kg | 0.639 kg | 一致 ✓ |
| Calf | 0.207 kg | 0.207 kg | 一致 ✓ |

**说明**: MJCF 模型已修正为与 URDF 一致的质量参数，确保两种格式的动力学行为相同。

### 10.2 关节定义差异

| 特性 | URDF | MJCF |
|------|------|------|
| 关节类型 | `<joint type="revolute">` | `<joint type="free">` + body hierarchy |
| 默认姿态 | 通过 mesh rpy | 通过 inertial quat |
| 摩擦力 | joint dynamics | geom friction |
| 阻尼 | joint dynamics damping | joint damping |

### 10.3 接触定义差异

| 特性 | URDF | MJCF |
|------|------|------|
| 接触几何 | collision sphere | geom sphere |
| 摩擦锥 | collision + material | geom friction |
| 触点检测 | contact sensor | touch sensor |

## 12. AI 迁移指南

### 12.1 从 Aliengo 到 Go2 的关键变更

```python
# 质量参数
go2_trunk_mass = aliengo_trunk_mass * 0.766  # 9.042 * 0.766 ≈ 6.92

# 惯性张量 (需要对角缩放)
go2_inertia = np.array([
    aliengo_inertia[0] * 3.2,    # Ixx
    aliengo_inertia[1] * 0.61,   # Iyy
    aliengo_inertia[2] * 0.14    # Izz
])

# 腿部长度
go2_hip_x = aliengo_hip_x * 0.806
go2_hip_y = aliengo_hip_y * 0.912
go2_thigh_length = 0.213  # 固定值
go2_calf_length = 0.213    # 固定值

# 力矩限制
go2_hip_torque_limit = 23.5  # 固定值 (原 44.4)
go2_thigh_torque_limit = 23.5  # 固定值 (原 44.4)
go2_calf_torque_limit = 45.0  # 保持不变

# 关节角度限制
go2_hip_limit = 1.0472  # ±60° (原 ±70°)
go2_calf_lower = -2.7227  # 保持
go2_calf_upper = -0.83776  # 扩大 (原 -0.646)
```
